# Cómo agregar el logo de la Unidad Educativa

## Pasos:

1. **Prepara tu logo:**
   - Formato recomendado: PNG con fondo transparente
   - Tamaño recomendado: 200x200 píxeles mínimo
   - Peso: Menos de 500KB

2. **Renombra el archivo:**
   - Nombre del archivo: `logo.png`
   - Si es JPG: `logo.jpg` (y actualiza en DashboardPage.js)

3. **Copia el archivo aquí:**
   - Ruta: `frontend/public/assets/logo.png`
   - Simplemente arrastra y suelta tu logo en esta carpeta

4. **El logo aparecerá automáticamente** en el dashboard cuando recargues la página

## Formatos soportados:
- PNG (recomendado para logos con transparencia)
- JPG (para fotos)
- SVG (para gráficos vectoriales)

## Ubicación final del logo:
El logo se mostrará en la esquina superior izquierda del Dashboard, junto al título "Dashboard - Tesorería 8vo C"
